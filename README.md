**浙江大学选课脚本**


**使用前请先安装python3和requests模块，推荐使用python3.75及以上版本运行**

**requests安装：pip3 install requests (pip install requests)**

这是一个帮助你自动选课的脚本，可以用于低分定点选课阶段的抢课

在浙大，选到优质课程可以很大程度改善你的学习体验；好的课程同时也对GPA提高有所帮助。
